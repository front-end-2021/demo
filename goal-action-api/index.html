**API**
  1. Get all Maingoal: get .../api/mains
  2. Get all Subgoal: get .../api/allsub
  3. Get all Action: get .../api/allaction
  4. Get Main by Id: get .../api/main?id=
  5. Get Sub: get .../api/sub?id=
  6. Get Action: get .../api/action?id=
  7. Add new Maingoal: post .../api/main 
    - body: {
              "Name" : "Name Goal",
              ["Description: "Des ...",]
              ["Budget" : 100,]
              ["Start: "20-Feb-2023",]
              ["End: "20-Feb-2023"]
          }
  8. Add new Subgoal: post .../api/sub 
    - body: {
            "Name" : "Name Goal",
            "ParentId" : "(uuid Maingoal)",
            ["Description: "Des ...",]
            ["Budget" : 100,]
            ["Start: "20-Feb-2023",]
            ["End: "20-Feb-2023"]
            }
  9. Add new Action: post .../api/action 
    - body: {
            "Name" : "Name Action",
            "ParentId" : "(uuid Subgoal)",
            ["Description: "Des ...",]
            ["ExpectCost" : 100,]
            ["TrueCost" : 100,]
            ["Start: "20-Feb-2023",]
            ["End: "20-Feb-2023"]
            }
  10. Update Goal: put .../api/goal/:id 
    - body: {
            "Name" : "Name Goal update",
            ["Description: "Des update ..",]
            ["Budget" : 100,]
            ["Start: "20-Feb-2023",]
            ["End: "20-Feb-2023"]
            }
  11. Update Action: put .../api/action/:id 
    - body: {
            "Name" : "Name Action update",
            "ParentId" : "(uuid Subgoal)",
            ["Description: "Des update ...",]
            ["ExpectCost" : 100,]
            ["TrueCost" : 100,]
            ["Start: "20-Feb-2023",]
            ["End: "20-Feb-2023"]
            }
  12. Get Actions by SubId: get .../api/actions?subid=
  13. Get Subs by MainId: get .../api/subs?mainid=
  14. Duplicate Subgoal: post .../api/copysub 
    - body: {
            "Name" : "Name Goal",
            "ParentId" : "(uuid Maingoal)",
            ["Description: "Des ...",]
            ["Budget" : 100,]
            ["Start: "20-Feb-2023",]
            ["End: "20-Feb-2023"]
            }
  15. Delete Main: delete .../api/main/id=
  16. Delete Sub: delete .../api/sub/id=
  17. Delete Action: delete .../api/action/id=
